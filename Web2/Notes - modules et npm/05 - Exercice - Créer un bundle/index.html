Moderniser le code du projet : Monster Monster!

Cet exercice vise à vous faire pratiquer npm et webpack en modernisant le code d'un projet déjà complété. Vous devez donc :

  1. Gérer le projet via npm
  2. Ajouter une dépendance vers webpack (+webpack-cli) et créer son fichier de configuration (webpack.config.js)
  3. Démarrer webpack via une commande npm afin qu'il construise un bundle à partir du fichier source principal (js/animation.js)
  4. Modifier le HTML afin qu'il insère uniquement le bundle au lieu de tous les fichiers JavaScript du dossier js.
  5. Modifier les fichiers du dossier js afin d'utiliser les imports/exports nécessaires au bon fonctionnement du projet.
