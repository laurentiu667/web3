Création d'un calendrier

Cet exercice permet de pratiquer l'intégration d'une librairie externe à un projet JavaScript

  1. Gérer le projet via npm
  2. Ajouter une dépendance vers webpack (+webpack-cli) et créer son fichier de configuration (webpack.config.js)
  3. Démarrer webpack via une commande npm afin qu'il construise un bundle à partir du fichier source principal (src/main.js)
  4. Modifier le HTML afin qu'il insère le fichier bundle au lieu du fichier src/main.js.
  5. Installer la librairie fullcalendar.io (voir section : Initialize a Calendar with Plugins)
  6. Dans votre fichier JavaScript et en suivant la documentation en ligne, ajouter un calendrier dans votre page Web. Prendre note que dans la documentation calendarEl fait référence à un noeud HTML que vous devez aller chercher via un querySelector. N'oublier pas d'ajouter cette ligne à la fin de votre code JavaScript : calendar.render();
